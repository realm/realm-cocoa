Group::getTable

(TightdbTable *)getTable:(NSString *)name;
(id)getTable:(NSString *)name withClass:(Class)obj;

This method will find and return a named table.

When no table class is specified, this function returns a dynamically typed table accessor. When a table class (MyTable) is specified as a template parameter, you get a statically typed table accessor.

It is an error to specify unmatching table class for an existing table in the group. Doing so will result in undefined behaviour. For this reason you should generally never call get_table() without first checking the type of the table with a call to has_table().

If no table exists with the specified name, a new empty table will be created.

Parameters:

name Name of the table you wish to look for.
obj Specifies a static type for the returned table accessor.

Returns:

A table reference object.
