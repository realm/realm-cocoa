These methods are used to insert new rows in a table.

WARNING: These methods support inserts with maximal performance. But they can easily corrupt the database if used incorrectly! When inserting a new row, you have to call the corresponding insert method for every column in the table and end the insert sequence by calling insert_done().

insert_binary() will copy the data provided.

Parameters:

ndx, row_ndx The column index.
ndx The row index.
value The value to insert.
data Pointer to start of binary data.
size Size in bytes of binary data.
